# StudioPro AI - Image Cartoonizer

A Streamlit-based application to cartoonize images using OpenCV and Pillow. Includes user authentication, processing history, and Razorpay integration.

## Setup Instructions
1. Clone the repository.
2. Create a virtual environment: `python -m venv venv`
3. Activate the virtual environment:
   - Windows: `venv\Scripts\activate`
   - Mac/Linux: `source venv/bin/activate`
4. Install dependencies: `pip install -r requirements.txt`
5. Run the application: `streamlit run main.py`

## Technologies
- **Frontend**: Streamlit
- **Processing**: OpenCV, Pillow
- **Database**: SQLite3
- **Payment**: Razorpay SDK
